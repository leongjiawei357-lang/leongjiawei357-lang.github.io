Online-Inventory-Document-System

How to run (local):

1. Open two terminals.

2. Start the server:
   cd server
   npm install
   node server.js
   Server runs at http://localhost:5000

3. Open the client:
   - Use VS Code Live Server to serve the client folder, or open client/login.html directly in the browser.
   - The frontend talks to the API at http://localhost:5000/api

  username: admin
  password: 1234
